# Blinkit---Sales---Dashboard
Interactive Blinkit Sales &amp; Delivery Dashboard built using Power BI. This dashboard provides insights into sales performance, customer orders, product categories, delivery status, and monthly sales trends through interactive visualizations and KPI cards.

## Overview
This Power BI dashboard analyzes Blinkit's sales and delivery performance using interactive charts, KPI cards, filters, and business insights.

## Features
- Total Sales Analysis
- Total Orders Tracking
- Total Customers Overview
- Average Order Value
- Monthly Sales Trend
- Sales by Product Category
- Top Selling Products
- Delivery Status Analysis
- Interactive Filters & Slicers
- Business Insights Section

## Tools Used
- Power BI
- Excel / CSV Dataset

## Dashboard Insights
- Identified top-selling products
- Analyzed monthly sales trends
- Compared sales across categories
- Tracked delivery performance
- Visualized customer order behavior

- Power BI Dashboard (.pbix)
- Dataset Files (.csv)
